Price Tracker Telegram Bot

Telegram-бот для отслеживания цен на товары с сайта DNS (Теперь не только в DNS).
Бот автоматически проверяет цену и уведомляет подписчиков о её изменении.

Возможности

📦 Подписка на уведомления о ценах

➕ Добавление товара по ссылке (только администратор)

❌ Удаление товара

📊 Отслеживание:

текущей цены

минимальной цены

максимальной цены


⏱ Автоматическая проверка цен по расписанию

🤖 Telegram-бот с кнопочным меню



📄 Конфигурация

Создать файл products.json в папке data

Создать файл .env:

BOT_TOKEN=your_token

ADMIN_ID=your_telegram_id

📦 Установка

Установить зависимости:

go mod download


Запустить бота:


go run main.go


⏱ Проверка цен


По умолчанию бот проверяет цены:


каждые 24 часа


Это можно изменить в scheduler.




📊 Данные

Все данные хранятся в:

data/products.json

🔒 Безопасность

только ADMIN_ID может:

добавлять товары

удалять товары
